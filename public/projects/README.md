# Project Images

Place your project screenshots/images in this directory.

## Image Naming Convention

Based on the project titles, use the following naming:

- `housing-authority-1.jpg` / `housing-authority-2.jpg` - Housing Authority - AI Lift
- `compliance-platform-1.jpg` - BOC & OCBC - Compliance Check Platform
- `icac-platform-1.jpg` / `icac-platform-2.jpg` - ICAC - Case Analysis Platform
- `esg-platform-1.jpg` - ESG Data Analytics Platform
- `ccas-platform-1.jpg` - CCAS Platform
- `green-hotel-1.jpg` - Green Hotel Training Game
- `data-visualization-1.jpg` - Multidimensional Data Visualization Technology
- `to-gather-1.jpg` - To Gather

## Image Requirements

- Format: JPG, PNG, or WebP
- Recommended size: 1200x675px (16:9 aspect ratio)
- Max file size: 2MB per image
- Images will be automatically optimized by Next.js Image component

## Adding Images

1. Add your project images to this directory with the naming convention above
2. The images will automatically appear in the project details page
3. You can add multiple images per project by adding `-2.jpg`, `-3.jpg`, etc.
